# Books-To-Scrape-Analysis
Web Scraping and Data Analysis project using Books to Scrape Website.

## Project Overview
This project focuses on collecting and analyzing book data from the Books to Scrape website using Python.

## Objectives
- Scrape book data from the website
- Clean and preprocess the data
- Perform Exploratory Data Analysis (EDA)
- Generate business insights
- Visualize key findings

## Tools Used
- Python
- Requests
- BeautifulSoup
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Workflow
1. Data Collection using Web Scraping
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis
4. Data Visualization
5. Business Insights and Recommendations

## Key Insights
- Analyzed book price distribution
- Compared ratings across books
- Identified category trends
- Evaluated stock availability
